разработай информационную систему под следующие требования: 
язык программирования - python 
субд - mysql, 
пользовательский интерфейс - telegram. 

предназначение системы: 
1) расчет балансов участников группы в telegram между начислениями за участие в тренировках и оплат вносимых на карту администратора группы. 
Для чего необходимо учитывать:
1) Управление составом участников группы telegram, информация об участнике группы обязательно должна содержать telegam id, и имя участника группы. Группу можно удалить (удалив все записи об участниках), создать из состава группы в telegram, по событию о добавлении участника в группу добавить участника в таблицу. 
2) Назначение администратора группы который будет наделен отличными от рядовых участников группы возможностями. 
3) Баланс за участие в тренировке вычисляется как разница сумм начисленных за участие во всех тренировках и сумм оплат когда либо вносимых; 
4) Система должна уметь сформировать в группе в telegram опрос с использованием функционала опросов имеющегося в telegram. Опрос должен содержать дату, время, место проведения тренировки, сумму за участие, на основании информации предоставленной администратором группы. Варианты ответов опроса: смогу, приду с другом которого нет в группе, не смогу прийти, пока не определился. 
5) Система должна вести учет фактов проведения тренировок, даты проведения, места проведения, и суммы за участие, на базе опросов проводимых среди участников группы; 
6) Система должна вести учет зарегистрировавшихся на конкретную тренировку участников группы в результате опроса. Список может пополняться, по факту голосования, участников, и  изменяться, в том случае если участник группы изменил свой выбор;
7) Система должна производить увеличение баланса каждого из участников группы, зарегистрировавшихся в результате опроса на фиксированную сумму стоимости оплаты одной тренировки, опубликованной в заголовке опроса, с указанием даты участия в тренировке; 
6) Система должна иметь возможность через интерфейс telegram от лица администратора группы или любого из участников группы сообщить от внесении оплаты за участие в тренировке (оплата может не совпадать с суммой за одну тренировку) вместе с суммой оплаты должна учитываться и дата оплаты. 
9) Каждый из участников группы должен иметь возможность узнать свой текущий баланс. Администратор группы должен иметь возможность узнать баланс каждого из участников группы
10) Каждый из участников группы должен иметь возможность получить расшифровку начислений и оплат. Администратор группы должен иметь возможность получить расшифровку баланса каждого из участников группы.
11) Система должна уметь по каждому из участников группы вести начальный баланс. Значение начального баланса может быть как положительным так и отрицательным и должно в расчете баланса использоваться по статье начисления. Все начисления и оплаты сделанные до даты начального баланса в расчет браться не должны. По каждому из участников группы начальный баланс может устанавливаться неограниченное количество раз, для расчета текущего баланса использовать значение с максимальной датой но предшествующей дате запрашиваемого расчета.   
12) Администратор группы должен иметь возможность через интерфейс telegram получить отчет о состоянии балансов каждого из участников группы.
13) Удаление участников из группы не должно приводить к удалению информации из учетной системы
Требуется создать скрипт создания базы данных и версию исходного кода на языке Python.
